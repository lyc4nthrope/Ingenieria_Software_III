# Components

Componentes UI reutilizables sin lógica de negocio.

## Estructura

- **ui/** - Componentes básicos atomizados (Button, Input, Modal, Card, Spinner, etc.)
- **layout/** - Componentes de estructura general (Navbar, Sidebar, ProtectedRoute, etc.)

## Naming Convention

- Los componentes deben usar PascalCase
- Los archivos deben ser `.jsx` y nombrados igual que el componente
- Cada componente debe exportarse como default y nombrado
